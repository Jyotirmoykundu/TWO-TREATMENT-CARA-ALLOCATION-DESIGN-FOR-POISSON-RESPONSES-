# TWO-TREATMENT-CARA-ALLOCATION-DESIGN-FOR-POISSON-RESPONSES-
This project explores a CARA (Covariate-Adjusted Response-Adaptive) design for allocating subjects between two treatments when the response follows a Poisson distribution (i.e., count data).

The idea is :
as data comes in, the allocation adapts based on past outcomes and covariates, so that better-performing treatments get more subjects over time.
The main goal is to assign more patients to treatments that are
clinically more effective, while accounting for individual covariates.

To make a correct decision about the treatment we will check which treatment results in less
number of recurrence of the disease. Based on this we will propose test statistics for a
demonstration purpose to select the better treatment.

To check the consistency of the proposed test procedures for testing the superiority of the
particular treatment over the other.
